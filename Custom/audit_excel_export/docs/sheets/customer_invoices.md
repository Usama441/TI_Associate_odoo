# Customer Invoices (Sheet)

This sheet is generated by the audit exporter with one block per invoice.

## Data source
- Report model: `account.report`
- Report ID: `audit_excel_export.audit_customer_invoices_report`
- Handler model: `audit.invoice.bill.report.handler`
- Base moves: customer invoices (`out_invoice`)
- Optional refunds: customer credit notes (`out_refund`) when `include_refunds = True`

## Filters applied (from wizard)
- Companies: `company_ids`
- Date range: `date_from` to `date_to` (on `invoice_date`)
- Journals: selected `journal_ids`
- Partners: selected `partner_ids`
- State scope (`invoice_bill_scope`):
  - `all_states` → `draft`, `posted`, `cancel`
  - `posted_only` → `posted`
  - `posted_cancelled` → `posted`, `cancel`
- Search term: report search bar value (`filter_search_bar`) when present

## Output behavior
- Each matched invoice is rendered as a separate two-section block:
  1. Invoice summary section:
     - `Invoice No`, `Invoice Date`, `Accounting Date`, `Currency`, `Customer`, `Due Date`, `Conversion Rate`
  2. Line details section:
     - `Label`, `Account`, `Quantity`, `Price`, `Taxes`, `VAT Amount`, `Amount`, `Currency`
- Line rows are generated from `move.invoice_line_ids` (ordered by line sequence and id).
- No sheet-level total row is appended.

## Notes
- Conversion rate uses `move.currency_rate` when available; falls back to a derived ratio from signed/company totals.
- Amount fields can be negative for credit notes/refunds.
